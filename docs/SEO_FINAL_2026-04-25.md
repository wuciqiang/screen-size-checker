## 任务二：内容优化

### /devices/standard-resolutions（最紧急：排名28）

**根本原因**：竞品页面有完整上下文解释，当前页面只有一张表格，内容深度不足。目标扩充至1500+词。

**建议H1/H2结构**：

```
H1: Screen Resolution Chart 2026: Complete Reference from 720p to 8K

H2: What Do These Resolutions Mean for You?
  → 按用途分段：网页开发 / 游戏 / 视频制作 / 日常办公

H2: Resolution vs. Monitor Size: What Looks Best?
  → 新增表格：分辨率 × 显示器尺寸 × PPI 联动
  → 示例：1080p最佳显示尺寸24"（约91 PPI），27"时仅82 PPI开始显糊

H2: CSS Breakpoints for Each Resolution
  → 对应媒体查询范围，直接给开发者复制代码

H2: Mobile Resolutions (360p to 1080p)
H2: Laptop Resolutions (HD to 4K)
H2: Ultrawide & TV Resolutions

H2: Common Questions
  → What resolution is best for gaming?
  → Is 1080p still worth it in 2026?
  → How to check your current resolution?
```

**内部链接**：→ 首页（检测当前分辨率）→ `/devices/compare` → `/hub/gaming-monitor-size-guide`

## 任务三：结构化数据（Schema）

### 当前状态

| 页面 | 现有Schema | 缺失 |
|------|-----------|------|
| 首页 | WebApplication ✓ | potentialAction |
| device页面 | Article ✓ | FAQPage、Dataset |
| hub页面 | **无** ✗ | Article、FAQPage |

### P0：hub页面加 Article + FAQPage Schema

在 `pages-config.json` 的hub页面配置中添加（以best-monitor-size-fps为例）：

```json
"structured_data": {
  "@context": "https://schema.org",
  "@type": "Article",
  "headline": "Best Monitor Size for FPS Gaming (2026): 24 vs 27 vs 32\"",
  "author": { "@type": "Person", "name": "Marcus Rivera" },
  "publisher": {
    "@type": "Organization",
    "name": "Screen Size Checker",
    "logo": { "@type": "ImageObject", "url": "https://screensizechecker.com/favicon.png" }
  },
  "datePublished": "2025-01-19",
  "dateModified": "2026-04-25"
},
"faq_structured_data": {
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "mainEntity": [
    {
      "@type": "Question",
      "name": "What is the best monitor size for FPS gaming?",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "24 inches is optimal for competitive FPS gaming. 78% of professional CS2 and Valorant players use 24-inch monitors for faster target acquisition and minimal eye movement."
      }
    },
    {
      "@type": "Question",
      "name": "Is 27 inch too big for FPS?",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "At 1080p, 27 inch can feel blurry. At 1440p, 27 inch is excellent for FPS. Sit 60-80cm away for optimal performance."
      }
    }
  ]
}
```

**实现方式**：在 `hub-builder.js` 中从Markdown的FAQ区块自动提取生成。

### P0：数据参考页加 Dataset Schema

在 `/devices/standard-resolutions`、`/devices/iphone-viewport-sizes`、`/devices/android-viewport-sizes` 部署 Dataset 类型，向Google表明这是结构化数据参考页：

```json
{
  "@context": "https://schema.org",
  "@type": "Dataset",
  "name": "Standard Screen Resolutions Reference Chart 2026",
  "description": "Complete reference of standard display resolutions from 720p to 8K, including pixel dimensions, aspect ratios, and common use cases.",
  "url": "https://screensizechecker.com/devices/standard-resolutions",
  "creator": { "@type": "Organization", "name": "Screen Size Checker" }
}
```

### P1：首页加 SiteLinksSearchBox

```json
"potentialAction": {
  "@type": "SearchAction",
  "target": "https://screensizechecker.com/?q={search_term_string}",
  "query-input": "required name=search_term_string"
}
```

### P2：验证 BreadcrumbList Schema

`show_breadcrumb: true` 的页面需确认 `templates/device-page.html` 中已输出 BreadcrumbList Schema。

## 任务五：中文页面（/zh/）快速提升

**现状**：11点击，CTR 8.8%（点击意愿极强），但展示量极少——Google几乎没有收录中文页面。

### 根本原因

1. 页面可能仍有 "Other language versions are being translated" 提示，Google不给未完成页面排名
2. hreflang配置缺失或错误
3. 中文内容为机器翻译，质量不足

### 操作步骤

**步骤1（最高优先级）：验证并修复 hreflang**

```bash
grep -r "hreflang" multilang-build/zh/ | head -5
```

如果缺失，在 `templates/base.html` 的 `<head>` 中添加：

```html
<link rel="alternate" hreflang="zh" href="https://screensizechecker.com/zh/" />
<link rel="alternate" hreflang="en" href="https://screensizechecker.com/" />
<link rel="alternate" hreflang="x-default" href="https://screensizechecker.com/" />
```

**步骤2：去掉"翻译中"提示**

检查中文页面是否有 "being translated" 相关提示文字，立即移除。

**步骤3：优化中文 Title**

```
首页中文 Title：
我的屏幕分辨率是多少？在线免费检测工具
```

**步骤4：加入中国品牌设备数据（差异化关键）**

在 `/zh/devices/` 下增加华为、小米、OPPO、vivo 的 viewport sizes 数据——这是英文版没有、中文用户有强烈需求的内容，也是中文页面相对英文页面唯一的差异化优势。

**步骤5：提交中文 Sitemap**

确认 `sitemap.xml` 包含所有 `/zh/` URL，在 GSC 中单独提交。
